# Shortcuts (MVP)

## Shell (global)
- `Ctrl + Alt + N` — focus NeuroBoost window (if enabled).

## Week Grid (Web)
- Mouse:
  - Drag empty space — create block (editor on release).
  - Drag block body — move (snap 15m).
  - Drag top/bottom edge — resize (snap 15m).
  - Double-click block — open editor.

- Keys (select a block first with a click):
  - `+` / `=` — nudge **+15m**
  - `-` / `_` — nudge **−15m**
  - `Enter` — open editor

Time display is MSK; storage is UTC.
